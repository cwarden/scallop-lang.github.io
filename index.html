Scallop, A Neuro-Symbolic Programming Language

Getting Started

Relational

Scallop is a language based on Datalog with additional support for negation and aggregation.

Scalable

Scallop's core language is implemented using Rust. In addition, we provide configurability over provenance information, allowing you to choose a differentiable reasoning module that suits your need.

Integratable

Scallop comes with a PyTorch binding for you to easily integrate a logical reasoning module in your existing PyTorch machine learning pipeline.
